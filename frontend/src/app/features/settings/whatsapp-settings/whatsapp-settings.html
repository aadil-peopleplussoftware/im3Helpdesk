WhatsApp Integration

Create tickets from WhatsApp messages automatically

1
Create a Twilio account and get a WhatsApp number from twilio.com
2
Set this webhook URL in your Twilio WhatsApp sandbox:
{{ webhookUrl }}
3
Fill your credentials below and save
Include 'whatsapp:' prefix as shown in Twilio

How it works

📱 Customer sends WhatsApp message to your number
🎫 Ticket auto-created in iM3 Helpdesk
👤 Customer auto-registered if new
🏷 Tagged as "whatsapp" automatically
↩ Agent reply goes back as WhatsApp message
